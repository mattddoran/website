Projects
Rally
IOS application visualizing GPS coordinates in Augmented Reality. Built on top of ARKit and MapboxKit on Swift with Xcode. Renders GPS Positions in AR as annotated nodes, syncing groups and GPS points to a Firebase DB. Decorates nodes with distance indicators and group avatars, and the nodes transform in 3D space and scale via distance.
Trump Drumpf Jump
A political satire game I made on iOS built on top of SpriteKit using Swift with Xcode. Within SpriteKit, used SKPhysics to move/manage sprite notes, as well as SKParticleEmitter to create particle effects. Stored player data locally using NSPlayerStats. Drew all sprites in Adobe Photoshop. Integrated leaderboards via GameCenter, and ads with Google Adwords.
3D Maze Explorer
Large computer graphics project made with OpenGL and C++. OpenGL was used for the graphics pipelines, allowing 3D transformations for models, camera, and scene. User interaction for movement and view switching between 3rd person and birds-eye view. Phong shader, with specular and ambient light support. Reads in .obj files to render scene objects.
High School Resource App
A template resource application for high schools. Purpose derived from a concept where students could view grades, faculty, agendas, announcements, etc. inside a hub. Designed to give smartphones a more practical purpose in high school settings. Scalable solution by creating a template application that could be contracted out to high schools. Designed using EachScape, a drag and drop app builder. Extensive interview process, developing a user persona on who would use this type of app.
